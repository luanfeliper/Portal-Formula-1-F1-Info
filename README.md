# Portal-Formula-1-F1-Info
Website interativo para explorar o universo da Fórmula 1, utilizando HTML, CSS e JavaScript. O site apresenta informações de calendário, pilotos, equipes, circuitos e resultados com dados organizados em JSON. O foco foi criar uma interface visualmente atrativa, com layout responsivo, navegação intuitiva e boa experiência em diferentes dispositivos.
